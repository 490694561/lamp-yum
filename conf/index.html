Congratulations. LAMP by yum is installed successfully!

Check environment: PHP Probe phpinfo phpMyAdmin 中文

phpMyAdmin Upgrade:

  • Execute script: ./update_phpmyadmin.sh

    • LAMP Uninstall:

  • Execute script: ./lamp.sh uninstall

    • Default Location:

  • Web root default location：/data/www/default

    • Process Management:

  • Apache：/etc/init.d/httpd (start|stop|restart|status)

  • MySQL：/etc/init.d/mysqld (start|stop|restart|status)

  • MariaDB：/etc/init.d/mysqld (start|stop|restart|status)

    • About:

  • 1、All of installation by yum，so update command is: yum -y update

  • 2、Apache is by base repository. MySQL or MariaDB and PHP is by Atomic repository

  • 3、If your VPS RAM ≥ 256M, recommended to compile and install LAMP, see LAMP stack installation scripts
